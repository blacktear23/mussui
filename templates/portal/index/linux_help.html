Usage

  1. Download client application
  2. Create directory at ~/muss and move muss-local to ~/muss
  3. Copy Configuration to config.json to ~/muss
  4. Execute client application: ~/muss/muss-local -c ~/muss/config.json
  5. Update system level SOCKS5 configuration: Server: 127.0.0.1, Port: 7070
