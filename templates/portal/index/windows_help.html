Usage

  1. Download client application
  2. Create directory at C:\muss and move muss-local.exe to C:\muss
  3. Copy Configuration to config.json to C:\muss
  4. Use cmd.exe to execute C:\muss\muss-local.exe -c C:\muss\config.json
  5. Update system level SOCKS5 configuration: Server: 127.0.0.1, Port: 7070
